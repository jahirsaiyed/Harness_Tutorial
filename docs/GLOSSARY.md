# Glossary — Harness Agent Tutorial

| Term | Meaning |
|------|---------|
| **Harness Agent** | The agent system built in this tutorial (`harness_agent` package). |
| **Agent harness** | Runtime around an LLM: tools, sessions, prompts, entry points. |
| **Tool registry** | Central register/dispatch for callable tools. |
| **Observation** | Structured tool result (`status`, `summary`, `next_actions`). |
| **Progressive disclosure** | Load skill metadata in prompt; full SKILL body on demand. |
| **Closed learning loop** | Post-turn step that may author or update skills. |
| **HARNESS_AGENT_HOME** | State directory (skills, memory, sessions, cron). |
| **REFERENCE_REPO_PATH** | Optional external clone for comparison reading only. |
